# Epileptic Seizure Prediction Using CHB-MIT

Seizure prediction project using the CHB-MIT scalp EEG database.

## Repository Structure

- `Data/` keeps small approved data files or data notes used by the project.
- `Feature_Engineering/` keeps preprocessing and feature extraction scripts.
- `Model/` keeps model training and evaluation scripts, plus model-specific result notes.
- `Outputs/` keeps generated summaries, plots, reports, or other saved outputs.

Large raw EEG files and generated intermediate datasets should stay outside the repository unless they are small enough and safe to publish.
